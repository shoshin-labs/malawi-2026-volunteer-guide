# Malawi 2026 Volunteer Guide

A public, shareable static guide for the Jersey Overseas Aid and Habitat for Humanity Malawi 2026 volunteer team.

Live site: this repository is published with GitHub Pages by the `Deploy GitHub Pages` workflow on every push to `main`.

## Local preview

```bash
cd site
python3 -m http.server 4173
```

Then open <http://127.0.0.1:4173>.

## Notes

This public repository intentionally contains only the website files and public-safe copy. Private source documents, payment details, and internal notes are kept out of the published repo.
