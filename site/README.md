# Malawi 2026 Volunteer Guide Website

Static website source for the Malawi 2026 volunteer guide.

## Files

- `index.html` - one-page guide
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and local checklist persistence

## Local preview

From this directory:

```bash
python3 -m http.server 4173
```

Then open <http://127.0.0.1:4173>.

This is suitable for GitHub Pages or any static host. If publishing publicly, keep private contact/payment details out of the page unless explicitly approved.
